CareerOS is a gamified job application tracker and portfolio system. Its purpose is to help users organize job applications, visualize progress, and present a professional portfolio with selectable UI themes (including a VS Code-inspired style).

## What It Does

- Tracks job applications across stages (applied, screening, interview, offer, rejected, ghosted).
- Provides analytics on application outcomes and response times.
- Shows a public portfolio with projects, skills, and a skill radar chart.
- Supports industry-based themes for the portfolio experience, including a VS Code-inspired option.
- Provides a VS Code-styled job tracker interface for application management.
- Includes a resume scanner that compares a resume against a job description for keyword fit.

## System Purpose

CareerOS centralizes the job hunt into one place: manage applications, improve outcomes with analytics, and showcase skills and projects to employers through a distinctive, themeable portfolio.
